# 我所理解的创业
My growth about read《how to start a startup》

## 警告：并非人人都适合创业
>一个人一生总要做一件不同寻常的事情

## 个人经历

2021年的12月中旬，读到李笑来的《YC创业笔记》，这是我读到的他的最后一本书，也是最令人震撼的书。

我于2011年研究生毕业，那一年是移动互联网元年，经过痛苦的择业过程，我选择了离家近的一家发电企业，国企，图个稳定。

十年之后，我依然是一名普通员工，从未拿过任何先进，因为没有一颗上进的心，浑浑噩噩过了多年，意识到不能再这样，穷则思变。

十年可以做很多事情，我却只读了几本书，很多是垃圾，也在情理之中，人在弱小的时候，选择能力也是弱的，同时生活能力，财务能力统统落后。幸亏从得到APP处知道了李笑来，进而了解到区块链，紧接着受其影响，开始改变。

多年后在读到《枪炮，病菌和钢铁》的时候，才知道自己的落后不是从2011年开始的，而是从再上一个十年，甚至上一个20年开始的。

戴蒙德在书中提出了**环境差异论**，否定了**种族差异论**。

我生于农村，从小喜欢读书，资质一般但学习从没有落下，赶上时代的浪潮，被推着步入了城市。

可思想依然停留在农村，那时候村里人的理想是不种地，家里人也是照着这个目标来培养我的，只要不种地，干什么都行。

什么，就是见识决定的，那时候工人就是一个好的选择，一个正式工厂的工人更是让人羡慕。

殊不知，社会变化太快，虚拟产业增速更快，而工业增速渐缓，从工资上就可以看出来，而且这个趋势依然在继续。

我结婚生子，无忧无虑地知足了几年，直到2016年开始醒悟，于是学习区块链，既然这是一个发展迅速的领域，那么我就可以借势而起。

一个人只要想改变，就会主动寻找机会，学会之后，我开始投资，开始讲课，开始接触更多的人，开始发现更多优秀的人，逐渐地发现自己的特长所在是这里，而不是工作中。

于是下定决心，全面转型。

## 目标

我没有从物理上接触过创业者，身边没有创业成功的人，没有和创业者交流过。

辞职的同事们没有创业的，至少我这么认为，他们只是换了一份工作。

我讲过一些人通过自媒体创业，一些人通过编程赚钱，一些人做社群运营，一些人做微商，这是我目光所及的普通人。

我见过有的人通过一个成型的作品受到青睐，获得了不错的机会。我见过有的人学习能力很强，通过自学打通了很多环节，不断地在新领域里探索。

我不知道什么叫创业，《小狗钱钱》里说要找到别人的需求，解决别人的问题。

我是带着困惑来听这门课，想从中获得启发的。

## 计划

这节课一共20节，每天一节，学完正好这一年就过完了，我也就完成了升华。

每听一节，记录下心得，模仿笑来的笔记，他带给我的震撼是，每一句话都被他重视起来，写出来的内容比原文还多。

这种感受在《鲍鹏山讲水浒》中也有体会，*鲁提辖拳打镇关西*中，一个字，一句话都可以洋洋洒洒几千言，分析得令人敬佩，这才叫读书。

要成为创业者，先要像个创业者，我没有核心的技能，也并非程序员，但自忖建立起了**成长型思维**，不会什么就学什么。

创业很难，在我看来是一个极其复杂的事情，我打小十分简单，不理世事，工作后也是远离杂务，多是听从指挥，这是不利之处。

我于区块链中看到了分布式去中心自组织，看到了主权个人，看到了剑走偏锋，思想的火苗时一旦被点燃，再也无法熄灭。

我尝试对别人讲起创业，也观察身边人的反应，他们固守一隅，认为这不可能。

我想，这才是我的机会，时代又一次垂青，再不振翅，即将老矣。
